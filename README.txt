This is a web application for searching news, viewing the course of cryptocurrencies (70 cryptocurrencies), after registration, the function of recording favorite news and a weather button appear. The weather rating of all the capitals of the world can be sorted by temperature and wind speed. The program is based on Java using the Spring framework. As a database used H2 for ease. Implemented spring security. Front end written in ntml using JS and Thymeleaf
